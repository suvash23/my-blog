Fast, unopinionated, minimalist web framework for [node](http://nodejs.org).

## Quick Start

Go to backend directory:

```console
$  cd /backend
```

Install dependencies:

```console
$ npm install
```

Start the server:

```console
$ npm start
```

View the website at: http://localhost:5000
